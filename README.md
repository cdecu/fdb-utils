Firebird fdbutils
=================
Firebird Utils 

Features
--------
-extract metadata
-generate metadata docs
-generate drop scripts
-generate json from table

Usage
-----
Use at command line::
```bash
fdbutils blabla
```

To use in Python::
```python
from ddlgenerator.ddlgenerator import Table
```
